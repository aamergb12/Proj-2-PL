# Proj-2-PL

This project implements a Prolog predicate find_exit/2 that solves a maze by generating a valid sequence of actions (up, down, left, right) leading from the start position to any exit.

Files

project2.pl — full solver implementation

devlog.txt — development log

Run using 

find_exit([[w,f,f],
           [s,f,e],
           [w,w,f]], A).
for example
